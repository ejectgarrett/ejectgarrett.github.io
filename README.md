# Bad Marks

# Installing
1. Download zip by clicking `Code > Download Zip` on this page
2. Extract anywhere of your choosing
3. Run `badmarks.exe` 

# Compiling
1. Ensure [Raylib](https://www.raylib.com) is properly installed
2. If on Windows, run `compile.sh` through MSYS2 MINGW64
3. If on Linux (and maybe MacOS), run `compilelinux.sh`
